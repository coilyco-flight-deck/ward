# ward-kdl agents aider

Exec-dialect CLI. Every verb runs `aider` with the granted subcommand (or its `argv` override) appended; the binary and its prefix are fixed and the caller can never substitute them.

## ward-kdl agents aider launch - interactive session (bare aider)

`aider`

Flags: unrestricted passthrough.

## ward-kdl agents aider headless - single non-interactive message; pass it as args

`aider --message`

Flags: unrestricted passthrough.
